# Environment (Окружение)

[English](README.md) | [Русский](README.ru.md)

---

Компоненты для **поверхностей и платформ** в духе Environment в Corgi Engine. Расширяют **SurfaceModifier** Corgi, добавляя поведение (например запрет прыжка) при сохранении базового трения и силы. Также **ловушки**: спавн телеграфа по триггеру и опционально ломаемые ловушки (добавить **TrapHealth** — любой урон ломает).

---

## Файлы

| Файл | Описание |
|------|----------|
| **SurfaceModifierRestrictions.cs** | Наследует Corgi **SurfaceModifier**. Добавляется на платформу (Collider2D), чтобы ограничить персонажей на этой поверхности. **Allow Jump** (по умолчанию выкл) отключает прыжок через `PermitAbility(false)` пока персонаж на поверхности; разрешение восстанавливается при выходе и при отключении компонента. Базовое трение и сила от родительского класса по-прежнему применяются. Меню: *Gameplay → Surface Modifier Restrictions*. |
| **TrapZone.cs** | **Ловушка (зона)**: префаб телеграфа спавнится при срабатывании любого из трёх опциональных триггеров — **(1)** вход объекта с одним из **тегов триггера** в коллайдер (trigger), **(2)** **Trigger On Hit** — по ловушке нанесли урон (TrapHealth), **(3)** **Trigger On Destroy** — ловушка сломана (OnBroken). Для всех действует общая логика one-shot. Старт телеграфа задаётся префабом. Опционально ломаемая ловушка: добавить **TrapHealth** (любой урон ломает). Меню: *Gameplay → Environment → Trap Zone*. |
| **TrapHealth.cs** | **Здоровье ловушки** (наследует Corgi **Health**): передаёт весь урон в base, уведомляет **TrapZone** при попадании и при смерти (**OnBroken()**). Добавлять на тот же объект или дочерний (с коллайдером на слое удара); **DownStrikeResponse** — для отскока при ударе вниз. Меню: *Gameplay → Environment → Trap Health*. |
