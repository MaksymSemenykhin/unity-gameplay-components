# Location Randomizer (Рандомизатор локаций)

[English](README.md) | [Русский](README.ru.md)

---

Спавнит экземпляры комнат/пещер при загрузке уровня из префабов и опционально связывает их с воротами. Два режима: **Legacy** (шанс спавна на префаб, без ворот) и **BlueGate** (одна комната на ворота, случайный префаб без повтора, привязка к воротам).

---

## Режимы

### Legacy (Use Blue Gate Count = выкл)

- **Cave Entries**: список (префаб, шанс спавна 0–1).
- Для каждой записи компонент бросает один раз; при успехе спавнит один экземпляр этого префаба.
- Каждый префаб используется не более одного раза за запуск (без дубликатов типов комнат).
- Позиции берутся из **Spawn Positions** по индексу (0 = первая комната); отсутствующие индексы используют (0,0).
- Без привязки к воротам.

### BlueGate (Use Blue Gate Count = вкл)

- **BlueGates**: массив трансформов (ворота на основной карте). Каждым воротам соответствует ровно одна комната.
- **Cave Entries**: пул префабов комнат. Для каждого ворот префаб выбирается **случайно без повтора** (по принципу перемешивания). Таким образом, каждая комната — разный префаб.
- **Правило: количество префабов должно быть >= количества BlueGate.** Если префабов меньше, чем ворот, в лог пишется предупреждение и комнаты получают только первые (по количеству префабов) ворота; остальные — нет.
- Каждая комната спавнится в **Spawn Positions**[индекс]. В префабе комнаты должен быть дочерний объект с именем **Green Gate Object Name** (например `"GreenGate"`) с компонентом **Teleporter**. Компонент связывает этот GreenGate с соответствующим BlueGate (оба `Teleporter.Destination` задаются), так что вход в BlueGate переносит игрока в комнату, а выход из комнаты — обратно.

## Справка по Inspector

| Раздел | Поле | Описание |
|--------|------|----------|
| **Caves** | Cave Entries | Префабы + шанс спавна (Legacy) или пул префабов (BlueGate). BlueGate: количество >= длине BlueGates. |
| **Spawn positions** | Spawn Positions | Мировая позиция для каждой комнаты по индексу. Отсутствующие индексы = (0,0). Отображается в Gizmos. |
| **When to run** | Run Mode | OnAwake или OnCorgiLevelStart. |
| **Performance** | Spread Over Frames | Если вкл — спавн растягивается на несколько кадров. Caves Per Frame — сколько за кадр. |
| **BlueGate – GreenGate** | Use Blue Gate Count | Если вкл — одна комната на BlueGate, привязка к воротам. |
| | BlueGates | Трансформы ворот на карте (порядок = индекс комнаты). |
| | Green Gate Object Name | Имя объекта выхода внутри префаба комнаты (с Teleporter). |
| **Debug** | Log Load Time | Логировать количество и время спавна по завершении. |

## События и тайминг

- По окончании спавна рассылается **MMGameEvent(LocationsSpawnedEventName)**. Подписка через `MMEventListener<MMGameEvent>` и проверка `e.EventName == LocationRandomizer.LocationsSpawnedEventName` для логики после готовности комнат (например, включение геймплея, построение навигации). Подписываться в OnEnable, отписываться в OnDisable.
- Можно вызывать **RunRandomization()** вручную из кода (например из GameManager); событие по-прежнему рассылается по завершении.

---

## Gizmos (вид сцены)

- При **Use Blue Gate Count** вкл: синяя сфера = позиция BlueGate, зелёная сфера = позиция спавна комнаты, линия между ними.
- В режиме Legacy: зелёные сферы = Spawn Positions.
